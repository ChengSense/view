基础语法

View.js 是一套构建用户界面的渐进式框架。与其他重量级框架不同的是，View 采用自底向上增量开发的设计。 View 的核心库只关注视图层，它不仅易于上手，还便于与第三方库或既有项目整合。 另一方面，当与单文件组件和 View 生态系统支持的库结合使用时，View 也完全能够为复杂的单页应用程序提供驱动。 如果你是有经验的前端开发者，想知道 View.js 与其它库/框架的区别，查看对比其它框架

插值 

    
    <span id="msg">Message: {{ msg }}</span>
    
    
    
    new View({
        view: "span#msg",
        model: {
            msg: "输入点什么?"
        }
    });
    
    
    
    输入点什么?

插值html片段 

    
    <span id="msg">Message: {{> show }}</span>
    
    
    
    new View({
        view: "span#msg",
        model: {
            show "<div>{{msg}}</div>"
            msg: "输入点什么?"
        }
    });
    
    
    
    输入点什么?

条件 @when 

    
    <div id="msg">
        @when(show,{
            <span>Message: {{ msg }}</span>
        })
    </div>
    
    
    
    new View({
        view: "div#msg",
        model: {
            msg: "输入点什么?",
            show : true
        }
    });
    
    
        
        输入点什么?

属性绑定

操作HTML元素的属性的是一个常见的需求，正如一切皆插值的理念一样的，View很简单的赋予你对元素属性操作的能力，然而这一切很简单。 让我们展示一个简单的例子，对元素属性绑定控制元素的隐藏和展示。 

    
    <span id="msg" style="display:{{show?'block':'none'}}">Message: {{ msg }}</span>
    
    
    
    new View({
        view: "span#msg",
        model: {
            msg: "输入点什么?"
            show: true
        }
    });
    
    
    
    输入点什么?

在{{}}指令里你可以使用“可以计算的表达式”，前提是要符合属性的规范，在规范里一切可以工作的很好

亦或是

    
    .show{
        display:block;
    }
    .hide{
        display:none;
    }
    
    
    
    <span id="msg" class="bg {{show}} {{some}}">Message: {{ msg }}</span>
    
    
    
    new View({
        view: "span#msg",
        model: {
            msg: "输入点什么?"
            show: "hide",
            some: ""
        }
    });

我们可以看到View如此简单的赋予你，超强的靠控制力，即使你的定义多个变量绑定的class也毫不费力； 上面的例子我们定定义了毫无用处的 class：some 然而并不影响最终的展示结果。
